Sobre este Proyecto

Este es un proyecto educativo diseñado para estudiantes que están aprendiendo desarrollo web.

Utiliza tecnologías modernas como HTML5, Bulma CSS Framework y JavaScript vanilla. El sistema de autenticación simula una base de datos real usando localStorage del navegador.
